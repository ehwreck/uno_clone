# UnoClone

An uno-like card base command line application written with elixir.

## Installation

The package can be installed by adding `uno_clone` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:uno_clone, "~> 0.1.0"}
  ]
end
```
